# AI Recipe Generator 🍳

A full-stack MERN application that generates recipes using AI based on user-provided ingredients.

## Features
- AI-powered recipe generation
- Ingredient-based search
- Recipe cards UI
- Favorites management

## Tech Stack
Frontend: React  
Backend: Node.js, Express  
Database: MongoDB  
AI: OpenAI API  

## Folder Structure
/client → React app  
/server → Backend API
